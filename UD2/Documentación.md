## Descripción de las Tareas

### Tarea 1: Instalación de un Controlador de Dominio Básico
En esta actividad se instaló un **controlador de dominio** utilizando **Windows Server**. Se configuró un dominio básico que permitirá administrar usuarios, equipos y políticas de seguridad dentro de la red local.

### Tarea 2: Creación de un pfSense para una Red Privada
En esta tarea se implementó un **pfSense**, que es un firewall/router de código abierto, para establecer una **red privada**. Esto permite segmentar la red, controlar el tráfico y gestionar de forma segura las conexiones entre los distintos dispositivos.

### Tarea 3: Convertir un Cliente del Dominio en Controlador Adicional
Aquí se tomó un **cliente existente del dominio** y se promovió para que funcione como un **controlador de dominio adicional**. Esto sirve para mejorar la redundancia y disponibilidad del dominio, así como para balancear la carga de autenticación.

### Tarea 4: Creación de Objetos de Dominio
En la última tarea se realizaron varias **creaciones de objetos dentro del dominio**, como usuarios, grupos y unidades organizativas. Esto permite organizar la estructura administrativa del dominio y aplicar políticas de manera más eficiente.
